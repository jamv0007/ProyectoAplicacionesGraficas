# ProyectoAplicacionesGraficas

Proyecto de la asignatura de Programación de aplicaciones gráficas

Una aplicación que permite:

+ Añadir y eliminar los modelos
+ Visualizar la escena con cámaras con movimientos pan, tilt, dolly, crane, orbit y un zoom
+ La escena tiene distintos tipos de iluminación y sombreado
+ Texturas y bomp mapping para aplicar a las texturas efecto de profudidad
+ Modo wireframe
